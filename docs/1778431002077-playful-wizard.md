## 6. Szczegóły Techniczne

### Typy danych

```typescript
// lib/pdf/types.ts
interface ReportData {
  generatedAt: Date
  properties: PropertyWithDefects[]
  companyName?: string
}

interface PropertyWithDefects {
  id: string
  name: string
  address: string
  defects: Defect[]
}

interface Defect {
  id: string
  title: string
  description?: string
  status: 'new' | 'in_progress' | 'done'
  photos: string[] // URLs from Supabase Storage
  createdAt: Date
}
```

## 10. Ryzyka i Mitigacje

| Ryzyko | Poziom | Mitigacja |
|--------|--------|-----------|
| **Duże PDFy > 10MB** | Średnie | Limit 50 usterek / PDF, podziel na części |
| **Konwersja zdjęć wolna** | Niskie | Cache base64 w localStorage |
| **Memory limit w przeglądarce** | Niskie | Worker thread dla dużych danych |
| **Font rendering issues** | Niskie | Test na Chrome/Firefox/Safari |
